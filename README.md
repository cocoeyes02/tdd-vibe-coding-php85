# TDD Vibe Coding with PHP 8.5

Claude Codeを使ったVibe CodingでTDD本をPHP 8.5で写経するプロジェクト

## 目的

cocoeyes02がClaude Codeを使ったVibe Codingへの理解度を深める

## 題材

Kent Beck著「テスト駆動開発（Test-Driven Development）」をPHP 8.5で実装する

- 本のコードはJavaとPythonで書かれているため、PHPに書き換えて実装する

## 習得したいClaude Code機能

以下の機能を使いこなせるようになることを目指す：

- コミット、issue作成・読み取り、調査
- TDDライクなコーディング、コードレビュー
- Slash Command
- skills
- sub agent
- hook
- MCP
- claude.md / rules

## 制約

- **Docker環境**で構築する
- **PHPUnit**、**Composer**を使用する
- **cocoeyes02が手動でコードを書くことは禁止**（すべてClaude Codeに書かせる）
